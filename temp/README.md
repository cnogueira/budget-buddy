# Quick Reference
## Database Setup
Execute via Supabase SQL Editor (in order):
1. `supabase/01_initial_schema.sql`
2. `supabase/02_dev_setup.sql`
## Run
```bash
npm run dev
```
## Test
1. Open http://localhost:3000
2. Click "+ New" in category field
3. Create a category (e.g., "Groceries")
4. Add a transaction
5. See colored category in transaction list
